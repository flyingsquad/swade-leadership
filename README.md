# swade-leadership
Leadership effects

Add the Apply Leadership action in the Leadership Actions compendium to characters who have Command, Hold the Line! or Fervor. Then use that action. Click the Apply button in the chat to apply those effects to allies, or Remove to remove them. Use the action whenever the character or allies may have moved out of range.

Similarly, add the Inspire action to a character with the Inspire Edge. Use that action in combat to bring up a dialog that will allow you to select the Trait that will benefit allies within range with a +1 bonus. This will last until the allies have taken their action.
